# Deep-Research Source Documents (Archive)

These three documents were generated by a separate Claude deep-research run and uploaded on 3 July 2026. They are archived here **for provenance only** — their validated content has been merged into the main documents in [`business/`](../README.md), and **where figures differ, the main documents supersede these files**.

Known deltas vs. the main docs:

- **FX rate:** these files assume ₱56 = US$1; the main docs use the verified ₱61.43 (3 July 2026), so all peso conversions here are ~10% low.
- **Rinse funding:** the "US$70M+ total" figure here was validated against Rinse's own Series D announcement and adopted (Tracxn's US$46.5M is noted as a discrepancy in the main doc).
- **CleanCloud pricing:** the US$99–325/mo figure here was validated (current 2026 pricing runs US$75–325/mo) and adopted over the older US$30–110 listings.
- **"No PH on-demand laundry apps":** partially corrected in the main docs — no *dominant* app exists, but small local services (Washwell, Lalaba, Laundrify PH) do operate; the GrabExpress/Messenger improvisation pattern stands.
- **Tracxn sector counts** (1,460 startups / 124 funded) could not be independently verified and are flagged as "as reported by Tracxn" in the main doc.
